# JavaLabs

Repositório com exercícios e pequenos programas em Java do módulo "Fundamentos".

Objetivo
- Reunir códigos práticos para estudar lógica, sintaxe e conceitos básicos de Java.
- Servir como espaço pessoal de aprendizado e experimentação — não é um projeto de produção.

O que tem aqui
- Programas simples (entrada/saída, operações, exercícios da disciplina) organizados em `src/fundamentos`.
