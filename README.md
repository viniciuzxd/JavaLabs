# JavaLabs

Repositório dedicado ao estudo e prática da linguagem Java, contendo exercícios, desafios e pequenos projetos desenvolvidos durante o aprendizado.

## 🎓 Cursos e Fontes

### Java COMPLETO: Do Zero ao Profissional + Projetos!
Os arquivos presentes no pacote `cursoudemy` fazem parte do curso **Java COMPLETO: Do Zero ao Profissional + Projetos!** ministrado por **Leandro Moura Leitão** na plataforma **Udemy**.

- **Pacote:** `cursoudemy.fundamentos`
- **Conteúdo:** Exercícios de lógica, operadores, estruturas de controle e desafios práticos do módulo de fundamentos.

## 📂 Estrutura do Projeto

O projeto está organizado da seguinte forma:

- **`src/cursoudemy`**: Contém os exercícios e desafios realizados durante o curso da Udemy.
  - `fundamentos`: Primeiros passos com a linguagem, variáveis, tipos primitivos, operadores e estruturas básicas.
- **`src/basico`**: Contém programas simples de entrada/saída, operações básicas e exercícios de lógica geral.

## 🚀 Objetivo
O foco deste repositório é:
- Reunir códigos práticos para consolidar o conhecimento em Java.
- Servir como um histórico pessoal de evolução no aprendizado da linguagem.
- Experimentar novos conceitos e bibliotecas de forma isolada.

## 🛠️ Tecnologias
- Java JDK (Recomendado 11 ou superior)
- IDE de sua preferência (VS Code, IntelliJ, Eclipse)
